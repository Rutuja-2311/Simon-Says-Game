# Simon-Says-Game
A web-based version of the classic "Simon Says" memory game. Developed using HTML, CSS, and JavaScript, this project includes: 1.  Dynamic color sequences generated each round 2. Score tracking and increasing difficulty  3. Responsive UI for desktop and mobile  Great project for practicing DOM manipulation, event handling, and simple game logic.

live link - https://rutuja-2311.github.io/Simon-Says-Game/
